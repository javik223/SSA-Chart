# Hooks

Custom React hooks for the Claude Charts application.

## Examples

- `useChartData.ts` - Hook for managing chart data state
- `useFileUpload.ts` - Hook for file upload handling
- `useChartConfig.ts` - Hook for chart configuration
- `useDebounce.ts` - Debounce hook for real-time updates
- `useExport.ts` - Hook for exporting charts
